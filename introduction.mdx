## Commands

PrismBoat has A LOT of commands, and I really mean, A LOT, check out some of the commands and how to use them below\!

<CardGroup cols="2">
  <Card title="Timeout Command" icon="clock" iconType="solid" color="#396cc9">
    \!timeout (user) (duration) \<reason\>

    \!timeout @mxybecookie 7d spamming
  </Card>
  <Card title="Remove Timeout Command" icon="clock" iconType="solid" color="#396cc9">
    \!removetimeout (user) \<reason\>

    \!removetimeout @mxybecookie idk
  </Card>
  <Card title="Purge Command" icon="trash" iconType="solid" color="#396cc9">
    \!purge (amount)

    \!purge 99
  </Card>
  <Card title="Setnick Command" icon="paintbrush" iconType="solid" color="#396cc9">
    \!setnick (user) (nickname)

    \!setnick @mxybecookie cookie
  </Card>
</CardGroup>
